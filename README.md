# Superstore Visualization

A Python-based data visualization project that analyzes and explores 
superstore sales data to uncover business insights through interactive 
and static charts.

## Project Overview
This project processes superstore retail data and generates 7 key 
visualizations to help understand sales performance, profit trends, 
and customer behavior across different regions and categories.

## Features
- Sales analysis by region, category, and sub-category
- Profit vs. Sales trend visualizations
- Customer segment breakdown
- Top-performing products analysis
- 7 automated visualizations saved as image files

## Technologies Used
- Python 3.7+
- Matplotlib
- NumPy
- Pandas
